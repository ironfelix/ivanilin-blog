# Contributing

Thanks for helping improve **astro-theme-aonote**.

## Development

```bash
git clone https://github.com/runsli/astro-theme-aonote.git
cd astro-theme-aonote
npm install
npm run dev
```

Requires **Node.js ≥ 22.12.0** (Node 22 LTS recommended).

## Pull requests

1. Fork the repo and create a branch from `main`.
2. Keep changes focused (one topic per PR).
3. Run `npm run check` and `npm run build` before opening the PR.
4. Describe what changed and why in the PR body.

## Reporting issues

Include:

- Node / npm versions
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if UI-related
